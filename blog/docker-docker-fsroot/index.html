еңЁдәҶи§ЈдәҶDockerе®№еҷЁзҡ„2дёӘеҹәзЎҖжҠҖжңҜ Linux namespace е’Ң Linux cgroup д»ҘеҗҺгҖӮжҲ‘д»¬жқҘиҒҠиҒҠDockerй•ңеғҸпјҢд№ҹе°ұжҳҜжҢӮиҪҪеңЁе®№еҷЁж №еҲҶеҢә/дёӢйқўзҡ„ж–Үд»¶зі»з»ҹпјҢд№ҹеҸ«еҒҡrootfs.

Linux rootfsвҖӢ

Linux rootfsпјҢLinux Root File System, Linux ж №ж–Үд»¶зі»з»ҹпјҢжҳҜеҢ…еҗ«еңЁе’Ңж №зӣ®еҪ•зӣёеҗҢзҡ„еҲҶеҢәдёҠзҡ„ж–Үд»¶зі»з»ҹпјҢжҳҜеҶ…ж ёеҗҜеҠЁж—¶жҢӮиҪҪзҡ„з¬¬дёҖдёӘж–Үд»¶зі»з»ҹгҖӮзі»з»ҹзҡ„еј•еҜјеҗҜеҠЁиҝӣзЁӢпјҢдјҡеңЁж №ж–Үд»¶зі»з»ҹжҢӮиҪҪд»ҘеҗҺпјҢжҠҠеҲқе§ӢеҢ–и„ҡжң¬е’ҢжңҚеҠЎзӯүеҠ иҪҪеҲ°еҶ…еӯҳдёӯиҝҗиЎҢгҖӮиҖҢеңЁDockerе®№еҷЁиҝӣзЁӢдёӯпјҢд№ҹжңүдёҖдёӘrootfsпјҢ иҝҷдёӘж–Үд»¶зі»з»ҹзҡ„дё»иҰҒеҠҹиғҪжҳҜз”ЁжқҘз»ҷе®№еҷЁиҝӣзЁӢжҸҗдҫӣйҡ”зҰ»еҗҺзҡ„жү§иЎҢзҺҜеўғгҖӮе®ғд№ҹжңүеҸҰеӨ–дёҖдёӘеҗҚеӯ—пјҢеҸ«еҒҡ Dockerе®№еҷЁй•ңеғҸгҖӮ
жңүдёҖдёӘжҜ”иҫғйҮҚиҰҒзҡ„зӮ№жҳҜпјҢrootfs еҸӘжҳҜдёҖдёӘж“ҚдҪңзі»з»ҹжүҖеҢ…еҗ«зҡ„ж–Үд»¶гҖҒй…ҚзҪ®д»ҘеҸҠзӣ®еҪ•пјҢе№¶дёҚеҢ…жӢ¬ж“ҚдҪңзі»з»ҹеҶ…ж ёгҖӮеңЁLinux ж“ҚдҪңзі»з»ҹдёӯпјҢж–Үд»¶зі»з»ҹе’Ңзі»з»ҹеҶ…ж ёжҳҜеҲҶејҖеӯҳж”ҫзҡ„пјҢж“ҚдҪңзі»з»ҹеҸӘжңүеңЁејҖжңәеҗҜеҠЁж—¶пјҢжүҚдјҡеҠ иҪҪжҢҮе®ҡзүҲжң¬зҡ„еҶ…ж ёгҖӮиҝҷе°ұж„Ҹе‘ізқҖпјҢеҗҢдёҖдёӘе®ҝдё»жңәдёҠйқўзҡ„жүҖжңүе®№еҷЁпјҢйғҪе…ұдә«е®ҝдё»жңәж“ҚдҪңзі»з»ҹзҡ„еҶ…ж ёгҖӮеҪ“е®№еҷЁиҝӣзЁӢйңҖиҰҒй…ҚзҪ®еҶ…ж ёеҸӮж•°гҖҒеҠ иҪҪйўқеӨ–зҡ„еҶ…ж ёжЁЎеқ—е’ҢеҶ…ж ёиҝӣиЎҢзӣҙжҺҘзҡ„дәӨдә’ж—¶пјҢе®һйҷ…дёҠйғҪжҳҜзӣҙжҺҘи°ғз”Ёе®ҝдё»жңәж“ҚдҪңзі»з»ҹзҡ„еҶ…ж ёгҖӮ

е®№еҷЁзҡ„ rootfsвҖӢ

е®№еҷЁзҡ„ rootfs иҮӘдёӢиҖҢдёҠеҲҶдёәдёүдёӘйғЁеҲҶпјҢеҲҶеҲ«жҳҜпјҢеҸӘиҜ»еұӮпјҲro+whпјүпјҢinitеұӮпјҲro+whпјүпјҢиҜ»еҶҷеұӮпјҲrwпјү

еҸӘиҜ»еұӮеҜ№еә” Docker е®№еҷЁзҡ„й•ңеғҸеұӮпјҢиҝҷйғЁеҲҶдёҚеҸҜиў«дҝ®ж”№
init еұӮеӨ№еңЁеҸӘиҜ»еұӮе’ҢиҜ»еҶҷеұӮд№Ӣй—ҙпјҢDocker е®№еҷЁеҚ•зӢ¬з”ҹжҲҗзҡ„дёҖдёӘеҶ…йғЁеұӮпјҢдё“й—Ёз”ЁжқҘеӯҳж”ҫ /etc/hostsгҖҒ/etc/resolv.confзӯүдҝЎжҒҜ
иҜ»еҶҷеұӮиҝҷдёҖеұӮзҡ„жҢӮиҪҪж–№ејҸжҳҜ rwпјҢдёҖж—ҰеңЁе®№еҷЁйҮҢеҒҡдәҶеҶҷж“ҚдҪңпјҢдҪ дҝ®ж”№еҗҺзҡ„еҶ…е®№е°ұдјҡеўһйҮҸең°еҮәзҺ°еңЁиҝҷдёӘеұӮдёӯ
