🔥 今日最值得关注机会（Top 3）​

1. Fundraisly —— AI 募资代理 🟢​

  • URL: https://fundraisly.com | PH: producthunt.com/products/fundraisly-4
  • 核心功能: AI 募资代理——自动找投资人、帮你约会议。官网称创始人平均 90 天内拿到 20-40 场投资人会议。
  • AI 模式: AI worker（事件驱动/自动执行）：事件="新投资人线索入库/触达跟进"，AI 自动研究、起草、发送 outreach、推进日程；人是审批例外（确认/拒绝某场会面）。
  • 竞争密度: 直接竞品很少见"专为 founder 募资"定位；相邻竞品 = Outreach(企业级销售外联, ARR 超 5 亿美元) + Apollo(130 万+ 用户, 联系人数据库) + Clay(AI 富化+外联, 2025 年爆发) + Instagiam(已死/被吞)；募资垂直没有单一强势玩家。
  • 窗口判断: 🟢还开着
  • 窗口原因: AI 外联工作流 2025 才成熟（LLM 能写出可信 outreach），90 天前还是"人写邮件+手动约"；现在窗口期在"把 Clay/Apollo 模式降维到 founder 募资场景"，1 年后会有垂直专用工具。
  • SEO 潜力: 高。核心词 fundraising automation、find investors for startup；长尾 how to find angel investors、pitch deck feedback tool、outreach to venture capitalists template、series angel investor list 2026。
  • MVP 难度: 中（外联文案+CRM+日历集成，LLM 占 70%，但"投资人名单数据"是硬成本，需 Apollo/ Crunchbase API 或自建抓取）。
  • 商业化潜力: 高。B2B、订阅制天然（按 outreach 量/预约会议数计费）；MRR 潜力 A 级（单客户高客单，founder 愿意为"拿到会面"付费）。
  • 差异化切角: 不做通用销售外联（被 Clay/Apollo 垄断），只做 founder → 投资人这一条窄链路，用 AI 把"找 + 写 + 约"串成 worker，人只看"要不要接这场会"。

2. LiveDemo —— 开源产品演示生成器 🟡​

  • URL: PH: producthunt.com/products/livedemo-4（官网未确认，tagline: "Open-source alternative to Storylane, Navattic, and Arcade"）
  • 核心功能: 录屏→自动生成可点击的产品演示/落地页，开源版对标 Storylane、Navattic、Arcade。
  • AI 模式: 偏 wrapper（人上传录屏→AI 生成可交互 demo），事件驱动成分低。
  • 竞争密度: Storylane(年收入数千万美元，已被 Google 收购前估值高)、Navattic(成熟 SaaS，客单价 $500+ /月）、Arcade、Productiv、Descript；开源切口中暂无强势玩家。
  • 窗口判断: 🟡快关闭
  • 窗口原因: 可交互 demo 是 B2B 标配已成共识，Storylane 类已被巨头觊觎；但"开源版"切中独立开发者/中小 SaaS 买不起 $500/月的人群，1 年窗口内可抢占。
  • SEO 潜力: 高。核心词 product demo software、clickable product walkthrough；长尾 open source product demo tool、navattic alternative、storylane alternative free、create product demo from video。长尾替代词是天然 pSEO 切口。
  • MVP 难度: 中-高（录屏解析+可交互 overlay 渲染，infra 中等）；但"开源"定位可借社区力量。
  • 商业化潜力: 中-高。B2B SaaS、按工作区/演示数订阅；独立开发者客单价低但量大，靠开源引流+企业版付费。
  • 差异化切角: 用"open source + 免费层"卡 Storylane/Navattic 的替代词流量（storylane alternative free），开源降低获客 CAC，企业版回收付费。

3. Pocket Screen —— Mac 浮动迷你屏 🟡​

  • URL: PH: producthunt.com/products/pocket-screen（tagline: "Keep any Mac window visible in a floating mini screen"）
  • 核心功能: 把任意 Mac 窗口钉成一个悬浮迷你窗口，永不遮挡主工作区（类似"常显小窗/画中画任意窗口"）。
  • AI 模式: 非 AI 工具，纯 macOS 系统级 utility（无 AI）。
  • 竞争密度: macOS 小工具类竞争低：相邻有 BetterTouchTool(付费 $10 一次性)、Rectangle(免费窗口管理)、Floating Window 类小众 App Store 产品；无专注"常显迷你窗"的强势品牌。
  • 窗口判断: 🟡快关闭（赛道本身小，但作为独立开发者练手/小额变现合适）
  • 窗口原因: macOS 系统级常显窗工具一直缺一个"极简、便宜、专注"的产品，独立开发者可在 1-2 周做出 MVP。
  • SEO 潜力: 中。核心词 macOS floating window、keep app always visible mac；长尾 macOS mini window app、picture in picture for any app mac、always on top mac app。
  • MVP 难度: 低-中（macOS Swift/Cocoa，系统 API 即可，1 周可出 MVP；infra 几乎为零）。
  • 商业化潜力: 低-中。个人开发者、App Store 单买 $5-10 或订阅 $2/月；单量小但几乎零成本，适合练手+小额睡后收入。
  • 差异化切角: 极致轻量+单一功能（只做"常显迷你窗"），用 App Store 长尾搜索获客，成本极低、retention 看口碑。

📐 今日发现的可复制模式​

1. "AI worker 替代销售/外联的某一段"模式（Fundraisly 类）：不是做全套 CRM，而是把"线索→研究→外联→约会议"中某一段变成 AI worker。成功要素：① 错误成本低（发错一封 outreach 无后果）；② 买单方=受益方（founder 自己省时间、自己出钱）；③ 可量化价值（"90 天内 20-40 场会面"）。同一模式换 domain（招聘 outreach、渠道合作、客户成功续费）即可复制。

2. "开源版 = 巨头免费替代层"模式（LiveDemo 类）：当某品类被 Storylane/Navattic 类高价 SaaS 垄断，"开源 + free 层"是独立开发者的黄金切口。靠替代词长尾流量获客（X alternative free），CAC 极低，企业版回收付费。

3. "macOS 单一功能 utility"模式（Pocket Screen 类）：纯系统级小工具，无 AI、零 infra、1 周 MVP、App Store 长尾搜索获客。不适合做主产品，但适合独立开发者练手 + 小额睡后收入 + 攒 App Store 口碑。

⛔ 今日不建议进入的方向​

  • Kimi K3 / 大模型平台（今日 PH 上榜，#3）：重 infra、巨头垄断、纯 hype，跳过。
  • Tencent EdgeOne Makers / Vapi / n8n / Replit / Supabase 类平台：已是成熟平台/巨头，无独立开发者切口。
  • Auriko（Trading desk for LLM calls）：金融 + infra 重 + 错误成本极高（money 领域），AI worker 约束 1 失败，不可进入。
  • Sim Studio / Clark Agent / ZooData 等 "AI agent framework / 数据层"：agent framework 赛道拥挤且 infra 重，跳过。
  • 纯聊天/纯 AI 增强界面（Orate、ElevenLabs 等）：本质是 CRUD + AI wrapper，不是 worker，难形成差异化。
  • Goldfish（Option 快捷键）：域名 goldfish.ai 已挂在 Spaceship 出售，且"键盘宏/快捷键"赛道饱和（Karabiner/Spark 已覆盖）。

🚀 今日最值得尝试的 MVP​

方向：AI 募资 outreach worker（Founder 专用版）

  • 最小功能：① 输入目标投资人名单（CSV/LinkedIn URL）→ ② AI 研究投资人（偏好、已投项目）→ ③ AI 起草个性化 outreach 邮件/DM → ④ 一键发送（集成 Gmail/Outreach API）→ ⑤ 自动跟进+约会议（Calendar API）。人只"审批是否发送"。
  • 推荐技术栈：Next.js(NextAuth) + Supabase(数据) + OpenAI/Anthropic(外联文案生成) + Apollo/ Crunchbase API(投资人数据) + Google Calendar/Gmail API(发送+约会议)。infra 极轻，月成本 < $50。
  • 开发时间估计：MVP 1-2 周（LLM 占 70% 工作量，核心是 prompt + API 集成 + 审批 UI）。
  • SEO 切入口：围绕 fundraising automation + find investors for startup + 长尾 outreach to venture capitalists template 做 pSEO 模板页（投资人 outreach 模板批量生成）。
  • 内容矩阵方向：AI 批量生成"行业 x 轮次"的 outreach 模板页（如 series-a-outreach-template-for-saas、angel-investor-email-for-fintech），可积累 1000+ Google 收录页面；多语言扩展（日语/德语 startup 市场）；适合 GitHub Actions 自动发布（模板页结构化、可批量）。
  • 商业化：Freemium（免费版 10 次 outreach/月）+ 订阅（$29-49/月，无限 outreach + 高级跟进）+ 可选 LD（$149 lifetime）。MRR 潜力 A 级。
  • 买单方对齐度：🟢 高对齐——founder 自己买、自己受益、省去自己的外联时间，无"替代员工"阻力。
